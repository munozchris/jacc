How has this course contributed to your education?

  • Enhanced my analytic skills and knowledge of syntactic topics across various languages
  • I felt like we covered a lot of syntax topics in this class, and I also got a better sense of whether I wanted to commit fully to a linguistics major.
  • Gave me an appreciation for syntax
  • I know some syntax now
  • Gave me a basic understanding of syntax to fulfill my basic major education.
  • I know about syntax in a non-generative way.
  • It really made me think about grammar in a way i never considered. Like, if you read the sentence "John stabbed Mike the knife" clearly it doesn't make sense, but it's hard to explain WHY it doesn't make sense-- and that's what this class teaches you to do. You might know transitive vs intransitive verbs, but have you ever thought about equi vs raising verbs? Why can you say "The cat seemed to be out of the bag" but not "The cat tried to be out of the bag?" ??? This class really broadened my knowledge of language, and I learned about things in a really enjoyable way.
  • I know what syntax is now.
  • Now I analyze syntax all the time.
